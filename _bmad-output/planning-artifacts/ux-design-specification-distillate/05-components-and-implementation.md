## Components and Implementation Strategy

- shadcn/ui coverage: Button (medium customization, larger targets+confidence colors); Card (high, confidence stripe+accordion); Badge (medium, confidence color mapping); Toast (low); Sheet (medium, bottom sheet mobile); Dialog (low, destructive only); Input (medium, numeric keypad+German locale); Select (low); Table (medium, sortable+inline status); Form (low); Skeleton (medium, custom shimmer); Tooltip (low); Tabs (low); Separator/Label (none); Dropdown Menu (low)
- 7 MVP-critical custom components:
- (1) PipelineHeader: nav+status element; shows lifecycle pipeline with real-time counts; stages array with id/label/count/icon; states: Default, Attention (Bereit pulse when count>0), Processing (shimmer), Empty, Tapped (105% scale+haptic); role=navigation aria-label="Rechnungs-Pipeline"; mobile abbreviates labels
- (2) AccordionInvoiceCard: core interaction component; collapsed=supplier+amount+confidence border (4px left); expanded=all fields+VAT+SKR+confidence per field+actions; swipe right=approve (green), left=flag (amber); swipe activation >20px, threshold 40% card width, below=spring snap 200ms, above=momentum 300ms ease-out; haptic at threshold; role=article; keyboard: Enter expand, Escape collapse, button alternatives to swipe
- (3) ConfidenceIndicator: per-field AI confidence; variants: dot (compact), badge (percentage), bar (progress); 95-100%=green checkmark no animation; 70-94%=amber triangle pulse 2s; 0-69%=red cross static; onTap opens source viewer; explanation text visible for amber/red
- (4) CameraCapture: full-screen viewfinder with document detection overlay; auto-capture when 4 corners stable >500ms; multi-capture mode with counter badge; gallery fallback for PDF/image; offline queue via IndexedDB+Service Worker; Camera API getUserMedia environment facing; max 2MB JPEG; open time <500ms; manual shutter 56px
- (5) SessionSummary: end-of-session card; shows invoice count, duration, estimated time saved, error count, streak weeks, export readiness; states: Perfect Session, With Corrections, Streak Milestone, Export Prompt (>=10 ready), First Session
- (6) TrustBadgeBar: ambient security; German flag + Gehostet in Deutschland + GoBD + DSGVO; fixed top 28-36px; Primary at 5% opacity; collapses to icon-only on scroll; never dismissable, never interactive
- (7) ExportAction: context-aware export button; states: Dormant (readyCount=0, text only), Available (1-9, subtle card), Prominent (>=10, Primary Light bg+pulse border), Month-End Urgent (last 5 days+readyCount>0)
- 5 post-MVP components: WeeklyRecapCard (Phase 2, after retention baseline); SupplierInsightBadge (Phase 2, after ~50 invoices); StreakTracker (Phase 2, professional tone milestones at weeks 4/8/12/26/52); SourceDocumentViewer (Phase 1.5, pinch-to-zoom 60fps); CorrectionEmailComposer (Phase 2, formal German business correspondence)
- State management: pipeline state=React Query/SWR server state; UI state=local React state; capture queue=IndexedDB via Service Worker; settings=server state cached locally
- Implementation roadmap: Phase 1 MVP (weeks 1-4) P0: CameraCapture, AccordionInvoiceCard, ConfidenceIndicator, PipelineHeader; P1: TrustBadgeBar, ExportAction, SessionSummary; Phase 1.5 (weeks 5-6): SourceDocumentViewer; Phase 2 (weeks 7-10): WeeklyRecapCard, SupplierInsightBadge, StreakTracker, CorrectionEmailComposer
